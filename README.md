# URL Shortener Project

## Project Description

This project is a full-stack URL shortener that allows users to create shortened links, track clicks, and analyze link usage over time. It emphasizes clean architecture, performance, and scalability. Users can optionally interact with a React-based dashboard to manage their links and view analytics.

## Tech Stack

* **Backend:** Node.js, Express
* **Database:** PostgreSQL
* **Caching & Rate Limiting:** Redis
* **Frontend:** React
* **Documentation:** Swagger/OpenAPI
* **Containerization:** Docker & Docker Compose

## Key Features

* Short URL generation with collision handling
* Click tracking and analytics
* Rate limiting and caching using Redis
* REST API endpoints for integration
* Optional React dashboard for user interaction
